Welcome to PickhardtPayments Package documentation!
======================================================

**PickhardtPayments** is a collection of classes and interfaces that help you to test and implement your dialect of
Pickhardt Payments into your on Lightning Application.

Check out the :doc:`readme` section for further information, including
how to `build and install` the project.

.. note::

   This project is under active development.
